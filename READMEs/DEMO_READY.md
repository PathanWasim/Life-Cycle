## 🐛 TROUBLESHOOTING

### Issue: Rate Limit Error
**Solution:** Restart backend (resets counter)
```bash
cd backend
npm start
```

### Issue: Page Not Loading
**Solution:** Hard refresh
```
Press: Ctrl + Shift + R
```

### Issue: Data Not Showing
**Solution:** Clear cache
```
Press: Ctrl + Shift + Delete
Select: "Cached images and files"
```

### Issue: Login Fails
**Solution:** Copy-paste credentials (case-sensitive)

### Issue: Blockchain Pending
**Explanation:** "This is normal - transactions are queued when MATIC is low. The system works perfectly without blockchain confirmation."
